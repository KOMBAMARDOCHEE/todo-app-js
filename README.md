#  Todo App — JavaScript Moderne

##  Description
Cette application Todo permet à l’utilisateur d’ajouter, afficher et supprimer des tâches dynamiquement.  
Elle met en œuvre les fondamentaux du JavaScript moderne (ES6+) et la manipulation du DOM sans framework.

##  Objectifs techniques
- Manipulation dynamique du DOM
- Gestion des événements utilisateur
- Logique applicative claire
- Code lisible et maintenable

##  Technologies utilisées
- HTML5
- CSS3
- JavaScript (ES6+)

##  Fonctionnalités
- Ajouter une tâche
- Supprimer une tâche en un clic
- Interface simple et intuitive

##  Lancer le projet
1. Cloner le dépôt ou télécharger les fichiers
2. Ouvrir le fichier `index.html` dans un navigateur
